# 🎵 Spotify Music Analytics Dashboard | Power BI

## 📊 Project Overview
An end-to-end data analytics project built on Spotify music data, designed to uncover insights about artist performance, song popularity, and streaming trends through an interactive, Spotify-themed Power BI dashboard.

---

## 🎯 Objective
To analyze what makes songs and artists successful on Spotify by examining popularity scores, chart hits, album types, and temporal trends across hundreds of artists and tracks.

---

## ⚙️ Tools & Technologies
| Tool | Purpose |
|------|---------|
| **Power BI** | Dashboard design, DAX measures, interactive visuals |
| **DAX** | KPI calculation, dynamic measures |
| **Excel / CSV** | Raw data preparation and cleaning |

## 🧩 Dashboard Structure (4 Pages)

### 🏠 Home — Navigation landing page
### 📈 Overview
- 789 distinct songs | 342 artists | Avg popularity: 90 | Avg duration: 3.28 min
- Songs by album type (Singles: 269, Albums: 562)
- Explicit vs Non-Explicit content split (17K vs 11K)
- Monthly popularity trends and distinct song counts by month
- Peak activity: **October (136 songs)**

### 🎤 Artists
- Taylor Swift dominates with **85 distinct songs** and **16M+ popularity score**
- Songs Hit Per Artist (Position 1 hits): Lady Gaga (77), Jung Kook (69), Sabrina (51)
- Artist-level breakdown: Max popularity, Avg duration, Avg popularity
- Historical data tracing back to 1942 (Bing Crosby & Fred Astaire)

### 🎵 Songs
- Top songs by popularity: *I Wanna Dance With Somebody* (51K), *Cruel Summer* (50K)
- Position 1 hits per song: *Die With A Smile* (77), *Seven* (75)
- Drill-down table: album type, position hits, album count, avg tracks per album

---

## 📈 Key Insights
- 💚 **Taylor Swift** leads all artists in distinct song count (85) and total popularity
- 📅 **October** recorded the highest distinct songs (136), suggesting seasonal release trends
- 🏆 **Albums** outperform singles in volume (562 vs 269) but singles have higher avg popularity (35.82%)
- 🎶 **"I Wanna Dance With Somebody"** tops popularity charts at 51K
- 📊 Popularity dips mid-year (May–Jul) and spikes in Q4, useful for release strategy planning
